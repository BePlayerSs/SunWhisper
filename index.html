FAQ rapido

Lo importante para que no se sienta incompleta.

Esta pagina ya esta lista para GitHub Pages?

Si. Solo necesitas subir el contenido de esta carpeta al repositorio de tu pagina y GitHub Pages la publica como un sitio estatico.

Puedo cambiar colores, textos o secciones despues?

Si. Esta hecha con archivos sencillos para que luego podamos editar rangos, meter nuevas categorias o conectar la tienda real sin romper el estilo.

Que falta personalizar ahorita?

Tu enlace real de Discord, cualquier enlace de tienda cuando exista y, si quieres, imagenes o capturas de cofres para que la pagina luzca todavia mas premium.
